## Number row of characters
#### !@#$%^&*()
* ! - Exclamation Point
* @ - Commercial "at" Symbol
* \# - "Hashtag" or Pound
* $ - Dollar Sign
* % - Percentage Symbol
* ^ - "little hat" - I only know this in French - "Accent Circonflex"
* & - Ampersand
* \* - Star
* ( - Left Parentheses
* ) - Right Parentheses
